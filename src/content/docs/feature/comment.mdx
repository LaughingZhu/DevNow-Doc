支持文章评论，目前只支持 Giscus 评论，可以在配置文件中修改，具体内容在 `src/components/Comments.astro` 中进行配置。


### 评论开关配置

```ts title="src/config/index.ts"
export default {
  ...
  // 默认开启
  giscus: true,
};

```

### 环境变量配置

具体信息获取可以参考 [Giscus](https://giscus.app/zh-CN)

在 Vercel 上配置环境变量 `PUBLIC_GISCUS_REPO_ID`、`PUBLIC_GISCUS_REPO`、`PUBLIC_GISCUS_CATEGORY_ID` 即可。

- PUBLIC_GISCUS_REPO_ID : Giscus 仓库 ID
- PUBLIC_GISCUS_REPO : Giscus 仓库名称
- PUBLIC_GISCUS_CATEGORY_ID : Giscus 分类 ID
