### 开关配置

```ts title="src/config/index.ts"
export default {
  ...
  // 默认开启
  search: true,
};

```

目前是基于开源框架 [lunr](https://github.com/olivernn/lunr.js) 实现的搜索，在运行时会动态进行分词，生成索引信息:

- search-docs.json
- search-index.json

搜索展示：

![](https://r2.laughingzhu.cn/02dd324dbbc76b200cdb07e197819522-5ec9b3.webp)


如果搜索服务无法正常使用，可以参考文章 [Search with Lunr](https://www.laughingzhu.cn/posts/search-with-lunr)。


目前的实现相对简单，分词库对中文支持也不是很好，后续会增加一些更好的搜索服务支持。
