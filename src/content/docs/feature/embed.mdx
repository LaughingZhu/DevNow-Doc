为了你的网站或博客有更好的体验，现在我们支持更多的媒体文件。你可以直接在网站中嵌入视频文件了，让您的网站内容更加丰富起来。

## 已支持

### Youtube

```mdx title="demo.mdx"
https://youtu.be/BAzA6fPkQjE?si=SrET_PGPmeajwSRw
```

import { YouTube } from 'astro-embed';

<YouTube id="BAzA6fPkQjE" />

### Vimeo

```mdx title="demo.mdx"
https://vimeo.com/974477342
```

import { Vimeo } from 'astro-embed';

<Vimeo id="32001208" />

### Links

```mdx title="demo.mdx"
https://www.laughingzhu.cn/posts/hello-world
```

import { LinkPreview } from 'astro-embed';

<LinkPreview id="https://www.laughingzhu.cn/posts/hello-world" />

### X (Twitter)

```mdx title="demo.mdx"

https://x.com/astrodotbuild/status/1511750228428435457
```

import { Tweet } from 'astro-embed';

<Tweet id="https://x.com/astrodotbuild/status/1511750228428435457" />

### Bluesky

```mdx title="demo.mdx"
https://bsky.app/profile/astro.build/post/3lcgawc2gp22x
```

import { BlueskyPost } from 'astro-embed';

<BlueskyPost id="https://bsky.app/profile/astro.build/post/3lcgawc2gp22x" />

### BaselineStatus

```mdx title="demo.mdx"
import { BaselineStatus } from 'astro-embed';

```

import { BaselineStatus } from 'astro-embed';
